## 🚀 Deployment Guide (GitHub Pages)

### Option A — Deploy from GitHub UI

1. Fork or upload this repository to your GitHub account
2. Go to **Settings** → **Pages**
3. Under *Source*, select `main` branch and `/ (root)`
4. Click **Save** — your portfolio will be live at `https://yourusername.github.io/repo-name`

### Option B — Deploy via Git CLI

```bash
# Clone the repo
git clone https://github.com/yourusername/portfolio.git
cd portfolio

# Make sure all assets are in the assets/ folder
ls assets/

# Push to GitHub
git add .
git commit -m "Deploy portfolio"
git push origin main

# Enable GitHub Pages in repo settings
```

### Option C — Custom Domain

1. Add a `CNAME` file to the root with your domain (e.g., `josephbernard.dev`)
2. Configure DNS with your domain registrar to point to GitHub Pages
3. Enable **Enforce HTTPS** in the Pages settings
